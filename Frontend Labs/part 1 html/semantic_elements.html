HTML Semantic Elements

HTML semantic elements clearly describe their meaning in a human- and machine-readable way. Examples include <header>, <footer>, and <nav>.

Example:

        
            <header>
                <h1>Welcome to My Website</h1>
            </header>
            <nav>
                <a href="/home">Home</a> |
                <a href="/about">About</a> |
                <a href="/contact">Contact</a>
            </nav>
            <footer>
                <p>© 2023 My Website</p>
            </footer>

Rendered Output:

Welcome to My Website

Home | About | Contact

© 2023 My Website
