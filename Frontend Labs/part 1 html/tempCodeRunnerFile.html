HTML Tables

HTML tables allow you to arrange data into rows and columns. The basic structure of a table includes the <table>, <tr>, <th>, and <td> tags.

Example:

        
            <table>
                <tr>
                    <th>Name</th>
                    <th>Email</th>
                </tr>
                <tr>
                    <td>John Doe</td>
                    <td>john@example.com</td>
                </tr>
            </table>

Rendered Output:

Name Email
John Doe john@example.com
